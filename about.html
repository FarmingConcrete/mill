This is the Farming Concrete Mill, where you can download data stored in the Barn. The data was collected with the Farming Concrete Data Collection Toolkit, developed by the Design Trust for Public Space in collaboration with Farming Concrete.

Privacy Considerations

We have tried to balance releasing useful data with the privacy of the gardens and those who helped to collect this data. As a result, we generate random, unique identifiers for each garden, gardener, and other identifying information. These identifiers are generated each time you download data, so they will not match across multiple queries. If you download one spreadsheet containing records from multiple records, the identifiers will match across metrics.

Where available, we provide the city, state, and zip code for gardens.

If you have questions about Mill or see data you believe is sensitive please let us know.
